## Featured Projects

### Hamili — AI Personal Finance Tracker
A cross-platform finance app for Android and Web with a built-in AI assistant. People track spending, set budgets and savings goals, and ask an assistant built on Google Gemini about their money in plain language. I built and shipped the whole stack: a Flutter frontend, a FastAPI backend with JWT auth and PostgreSQL, and a GitHub Actions pipeline that builds and signs the Android release. It runs live on Firebase Hosting and Render.

Technologies: Flutter, Dart, FastAPI, PostgreSQL, SQLAlchemy, Google Gemini API, Riverpod, GitHub Actions, Firebase Hosting, Render

### TUPVConnect
A campus communication and student services platform for the Technological University of the Philippines Visayas. It centralizes announcements, events, organizations, campus maps, and student resources in one place, with role-based access for students, faculty, and administrators.

Technologies: React, Node.js, MySQL, REST APIs, JavaScript, HTML, CSS
